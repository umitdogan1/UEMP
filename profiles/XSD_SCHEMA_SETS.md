# XSD Schema Sets (schemaSet Allowlist)

UEMP profiles may pin the underlying structural XSD layer using `config.schemaSet` in an `xsd` stage.

This document lists the currently allowlisted `schemaSet` values used by the reference implementation.

## Allowlisted `schemaSet` values

- `IATA-NDC/21.3`
- `UBL/2.1`
- `CII/D16B`
- `ISO20022/MR2019`
